02. Tentang Saya

Saya adalah lulusan S1 Teknik Informatika dari Fakultas Ilmu Komputer, Universitas Muslim Indonesia dengan IPK 3,96, yang memiliki minat dan pengalaman sebagai junior web developer, spesialis basis data, serta data scientist.

Saya berpengalaman mendukung pengembangan riset dan teknologi informasi melalui perancangan sistem pakar klinis menggunakan metode Certainty Factor dan Forward Chaining, penerapan algoritma Machine Learning seperti Naive Bayes, pengelolaan komputasi paralel, serta full-stack web development menggunakan PHP dan MySQL.

Memiliki kemampuan analisis data yang baik, teliti, disiplin, serta mampu bekerja secara terstruktur baik secara individu maupun dalam kolaborasi akademik untuk mendukung publikasi riset dan efektivitas pengembangan sistem secara profesional.
